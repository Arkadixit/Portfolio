# Images Folder

Place your project images here.

## Recommended Image Specifications

- **Format**: JPG or PNG (WebP for better compression)
- **Aspect Ratio**: 16:10 (matches the modal image container)
- **Resolution**: 800x500px minimum
- **File Size**: Keep under 200KB for fast loading

## Naming Convention

Use descriptive names that match your projects:
- `thesis-optimization-results.jpg`
- `unity-game-gameplay.png`
- `home-server-dashboard.jpg`

## Usage

Reference images in your project cards using the `data-images` attribute:

```html
data-images='["images/my-project-1.jpg", "images/my-project-2.jpg"]'
```
